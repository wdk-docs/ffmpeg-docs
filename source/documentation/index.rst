Community Contributed Documentation
----------------------------------------------------------

   :maxdepth: 2

- `An FFmpeg and SDL Tutorial <http://dranger.com/ffmpeg/>`_ by Stephen Dranger, explains how to write a video player based on FFmpeg.

Books about FFmpeg
----------------------------------------------------------

.. toctree::
   :maxdepth: 2

- `FFmpeg Basics <http://ffmpeg.tv/>`_ by Frantisek Korbel, describes various FFmpeg features and common tasks.
- `FFmpeg Beginner's handbook Chinese Version <http://book.chinaffmpeg.com/>`_ by Steven Liu, describes FFmpeg common use method in Chinese, from command line to API usage.
